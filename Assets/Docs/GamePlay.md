# ゲーム全体説明 / Game Logic Overview

本作は、実験場のような閉じた世界を舞台に、ネズミの主人公が爆弾を運びながら区画を進んでいく短編アクションパズルゲームである。

ゲームは全12ステージで構成される。各ステージは小さな独立区画として作られており、プレイヤーは区画内に配置された爆弾を使って、道を塞いでいる段ボール製の障害物を破壊し、次の区画へ進む。

基本的な目的は単純である。

* 爆弾を見つける
* 爆弾を目的地まで運ぶ
* 段ボールの障害物に爆弾をぶつける
* 爆発で道を開く
* 次の区画へ進む

ただし、爆弾を持っている間、主人公の身体能力は低下する。移動速度が落ち、ジャンプ力も弱くなるため、爆弾を持ったまま即興で進むのは難しい。プレイヤーは爆弾を拾う前に、どの道を通るか、どの足場を使うか、どこでジャンプするかをある程度考えておく必要がある。

本作の中心となる遊びは、反射神経よりも「爆弾を持つ前のルート確認」にある。
爆弾を持った後に考え始めるのでは遅い。ステージの構造、ギミックの性質、移動制限を見たうえで、安全かつ最短に近い運搬ルートを組み立てることが重要になる。

## 基本ルール

### 主人公

プレイヤーはネズミの主人公を操作する。
主人公は通常状態では自由に移動・ジャンプできるが、爆弾を持つと移動能力が下がる。

爆弾を持っていない状態では、ステージを観察し、ルートを確認することが主な役割になる。
爆弾を持っている状態では、制限された身体能力の中で、事前に考えたルートを実行することが主な役割になる。

### 爆弾

爆弾は各危険区画に配置される主要ギミックである。
プレイヤーは爆弾を持ち運ぶことができる。

爆弾は段ボール製の障害物を破壊するために使う。
爆弾を持った状態で障害物へ到達し、適切にぶつけることで爆発が起こり、次の区画へ進む道が開く。

爆弾は便利な道具であると同時に、危険物でもある。
爆弾を持っている間は移動能力が落ちるため、通常時には簡単に通れる道でも、爆弾を持った状態では難しくなる場合がある。

### 段ボールゲート

各ステージの出口や重要な通路は、段ボールで塞がれている。
通常の移動では突破できず、爆弾による破壊が必要になる。

段ボールゲートは、ステージクリアの明確な目標である。
プレイヤーは爆弾を段ボールまで運ぶことで、区画の出口を開く。

### ステージ構造

ステージは全12個の小区画で構成される。
各ステージは独立した課題を持つが、全体としては少しずつ難易度が上がっていく。

序盤では、爆弾を持つ、運ぶ、段ボールを壊すという基本動作を学ぶ。
中盤では、ジャンプ力低下や移動速度低下を前提にしたルート選択が重要になる。
後半では、環境ギミック、足場、危険地形、キャラクターから得た情報を組み合わせて進む必要がある。

### 環境ギミック

ステージには、移動を補助または妨害するギミックが配置される。

例として、踏むと跳ねるキノコのようなギミックがある。
通常のジャンプでは届かない場所でも、キノコを踏み台にすることで到達できる。
一方で、爆弾を持った状態ではジャンプ力が下がるため、同じギミックでも通常時とは使い方が変わる。

環境ギミックは、単なる装飾ではなく、爆弾運搬ルートを組み立てるための要素として機能する。

## ゲームループ

本作の基本的なゲームループは以下である。

1. ステージに入る
2. 地形と障害物を確認する
3. 爆弾の位置を確認する
4. 段ボールゲートまでの運搬ルートを考える
5. 必要であればキャラクターと会話し、ヒントや世界観情報を得る
6. 爆弾を持つ
7. 移動能力が下がった状態でルートを進む
8. 爆弾を段ボールにぶつけて破壊する
9. 次の区画へ進む

このループを12ステージ分繰り返しながら、プレイヤーは徐々に世界の構造やキャラクターたちの事情を知っていく。

## 会話と攻略ヒント

ステージ内には、特定のキャラクターが配置されていることがある。
キャラクターとの会話では、攻略のヒント、世界の仕組み、キャラクター同士の関係、外の世界に関する断片的な情報を得られる。

会話はステージ攻略に必須ではない場合もあるが、プレイヤーが世界を理解するための重要な要素になる。

主な役割は以下である。

* ステージギミックの使い方を自然に伝える
* 爆弾や区画のルールを説明する
* 「外」という言葉への違和感を提示する
* バニラやヤクモなど、主要キャラクターの価値観を見せる
* 終盤の目的地となる場所への手がかりを渡す

会話はチュートリアルを兼ねるが、単なる説明文ではなく、キャラクターの癖や関係性を通じて情報を出す。

## 世界の構造

この世界は、複数の区画に分かれた閉鎖環境である。
多くの住人は安全な区画に定住しており、危険な区画へ移動しようとはしない。

区画と区画の間には、食べ物が供給される場所が存在する。
住人たちはそれを「桃源郷」と呼んでいる。
桃源郷によって最低限の生活が保証されているため、多くの住人は外へ出ようとしない。

一方で、各危険区画には爆弾が出現する。
爆弾は住人たちにとって危険物であり、ほとんどの者は近づかない。
主人公だけがそれを運び、道を開いていく。

この世界には、「外」について語ることを避ける空気がある。
外は存在しないものとして扱われているわけではない。
むしろ、存在を考えること自体が不安や混乱を呼ぶため、多くの住人が話題にしたがらない。

## キャラクターとゲームロジックの関係

### バニラ

バニラは危険な場所や誰かが失敗した場面を絵に残すウサギの画家である。
彼女の絵は一見すると悪趣味に見えるが、結果的には危険箇所を示す目印にもなっている。

ゲーム上、バニラの絵はプレイヤーに対する視覚的なヒントとして機能する。
物語上は、彼女がこの世界で見たものを壁に残している痕跡でもある。

バニラ本人は、自分の絵を攻略ヒントとして描いているわけではない。
しかしプレイヤーは進行するにつれて、彼女の絵が単なる落書きや悪趣味な記録ではなく、この世界を歩くための手がかりになっていることに気づく。

### ヤクモ

ヤクモは猫の科学者である。
彼は爆弾の出現周期、桃源郷の供給間隔、環境ギミックの性質など、この世界の規則性を調べている。

ゲーム上、ヤクモはチュートリアルや攻略ヒントを与える役割を持つ。
ただし、単なる案内役ではない。
彼は世界の外側に興味を持ちながらも、自分の居場所や役割を失うことを恐れている。

ヤクモの情報は、ステージ攻略に役立つ実用的なものと、世界の謎に関わる不穏なものが混ざっている。

### バニラの父

バニラの父は、安全な区画で暮らす住人の価値観を代表するキャラクターである。
彼はかつて外にいた記憶を持っているが、その外は自由で美しい場所ではなく、冷たい金属のケージだった。

ゲーム上、彼は終盤へ向かうための手がかりを渡す役割を持つ。
また、バニラの絵に対する見え方を変える役割も持つ。

彼の会話によって、プレイヤーは「外に出ること」が単純な救済ではないことを知る。

## クリア条件

各ステージの基本クリア条件は、段ボールゲートを爆弾で破壊し、次の区画へ進むことである。

最終的には、誰も近づきたがらない場所、そして高い塔へ向かうことが大きな目的になる。
塔はこの世界のタブーに近い場所であり、外への手がかり、あるいはこの世界の仕組みに関わる何かがあると示唆される。

## 失敗条件

爆弾の扱いを誤る、危険地形に落ちる、ルート選択を間違えるなどした場合、プレイヤーは失敗する。
失敗後はステージ内のチェックポイント、またはステージ開始地点から再挑戦する。

本作では、失敗そのものも世界観の一部として扱われる。
バニラの絵が失敗の場面を描いているように、プレイヤーの失敗はこの世界に残る痕跡の一つとして意味づけられる。

## 設計意図

本作の中心は、複雑なパズルを解くことではなく、制限された状態で危険物を運ぶ緊張感にある。
爆弾を持つことで主人公は弱くなる。
弱くなった状態で進むためには、事前に道を見て、考え、覚えておく必要がある。

つまり本作は、瞬間的な操作技術よりも、観察と準備を重視するゲームである。

また、ゲームの進行と会話は分離していない。
キャラクターの会話は、攻略ヒントであり、世界観の断片であり、プレイヤーが先へ進む動機でもある。

爆弾を運んで道を開くこと。
危険な場所に残された絵を見ること。
科学者の記録を聞くこと。
安全な場所に留まる住人と話すこと。

それらを通じて、プレイヤーはこの世界が何なのか、なぜ外が禁じられているのか、自分がどこへ向かうべきなのかを少しずつ理解していく。
